# ⚡ Zenitsu Agatsuma — Pure CSS Art Challenge

A single-div inspired, hand-coded illustration of **Zenitsu Agatsuma** from *Demon Slayer*, built entirely out of semantic HTML and custom CSS shapes. This project contains absolutely no image files, SVGs, or external asset URLs.

## 🎨 Visual Features & Design Choices

To accurately humanize and depict Zenitsu using only code, several specific CSS techniques were used:
* **The Hair:** Crafted using blocky, absolute-positioned layers (`.bang`) paired with a `skewY` transform and an orange `border-bottom` to recreate his signature jagged, dyed-orange tips.
* **The Expression:** Features his iconic thick, blocky, rectangular eyebrows angled upward (`transform: rotate()`) to highlight his classic "worried/terrified" look.
* **The Haori (Robe):** Utilizes pure CSS triangle generation (using zero-width elements with transparent borders) scattered dynamically across his signature yellow coat.
* **The Eyes:** Layered ellipses utilizing subtle `linear-gradient` fills to represent his golden-brown eyes without relying on an editor tool.

---

## 🛠️ Built With

* **HTML5:** Semantic, clean structure mapping out facial and garment layers.
* **CSS3:** Custom shapes using `border-radius` manipulation, absolute layout coordinates, pseudo-elements (`::after`), and CSS transforms.
